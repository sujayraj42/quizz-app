# Contributing to Study-Buddy Quiz Room

Thank you for your interest in contributing! Here's how to get started.

## 🎯 How to Contribute

### 1. Fork the Repository
Visit [github.com/YOUR_USERNAME/study-buddy-quiz-room](https://github.com)
Click **Fork** button in top-right

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/study-buddy-quiz-room.git
cd study-buddy-quiz-room
```

### 3. Create a Feature Branch
```bash
git checkout -b feature/amazing-feature
```

### 4. Make Your Changes
- Edit files as needed
- Test your changes locally: `npm run dev`
- Follow the existing code style

### 5. Commit and Push
```bash
git add .
git commit -m "Add amazing feature"
git push origin feature/amazing-feature
```

### 6. Create Pull Request
- Go to your fork on GitHub
- Click "Compare & pull request"
- Write description of changes
- Submit pull request

## 📋 Contribution Guidelines

- Keep commits focused and descriptive
- Update README.md if adding features
- Test thoroughly before submitting
- Follow existing code formatting

## 🐛 Reporting Bugs

1. Check if bug already reported in Issues
2. Create new Issue with:
   - Clear title
   - Detailed description
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable

## ✨ Feature Requests

1. Open GitHub Issue
2. Tag with `enhancement`
3. Describe use case and benefits
4. Provide mockups if UI-related

---

Happy contributing! 🚀
